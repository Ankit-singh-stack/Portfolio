# Ankit Anand Singh - MERN Portfolio

A premium Full Stack Developer portfolio built with the MERN stack.

## Tech Stack
- Frontend: React.js, Vite, Tailwind CSS, Framer Motion
- Backend: Node.js, Express.js
- Database: MongoDB
- State Management: Context API
- Additional Libraries: React Router DOM, Axios, React Icons, React Toastify

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start frontend and backend together:
   ```bash
   npm run dev
   ```

## Backend config
Copy `backend/.env.example` to `backend/.env` and add your MongoDB URI and JWT secret.

## Features
- Dark/light mode toggle
- Sticky navigation and section highlighting
- Animated hero, skill cards, and timeline
- Contact form saving messages to MongoDB
- Admin APIs for messages and content management
- Responsive layout with glassmorphism and gradient accents
